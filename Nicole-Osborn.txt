 1. What is Semantic HTML? 
 Semantic HTML means it uses tags that describe the elements on a website.

    2. What is HTML used for? 
    HTML sets up the structure and content of a website.

    3. What is an attribute and where do we put it? 
    Attributes provide aditional information about an element. 

    4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is for the main header, and should only be used once on a page.

    5. Name two tags that have required attributes
    The <a> tag and the <img> tag require attributes.

    6. What do we put in the head of our HTML document? 
    The head contains information about the page, which should include a title, and sources of linked data like .js and .css files.

    7. What is an id? 
    An id is a name for a particular element.

    8. What elements can I add an id to? 
    I believe you can add an id to any element. 

    9. How many times can I use the same id on a page? 
    Only once.

    10. What is a class? 
    A class is given to elements which will share the same styling.

    11. What elements can I add a class to? 
    A class can be added to any HTML elements.

    12. How many times can I use the same class on a page? 
    You can use the same class on as many items as you need.

    13. How do I get my link to open in a new tab?
    Add a target attribute to the a tag.

    14. What is the alt attribute in the image tag used for? 
    To display alternate text if the image does not show up, and also allows screen readers to read the alt text for visually impaired users.

    15. How do I reference an id?
    #idname {}

    16. What is the difference between a section and a div
    Section defines a section of the webpage, where div tags have no semantic purpose.

    17. What is CSS used for? 
    Styling and positioning of the elements on the page.

    18. How to we select an element? Example - every h2 on the page
    h2{}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    A class can be used on multiple elements for which you want to use the same styling. For example, you might have a class for each of the main sections of a web page so that you can apply the same text, color, and spacing.
    An id is used to single out a particular element. If you wanted to style a particular image in one of the sections, you would use an id tag.

    20. How do we select classes in CSS?
    .class{}

    21. How do we select a p element with a single class of “human””?
    .human p{}

    22. What is a parent child selector? When would this be useful? 
    The ">" can be used to select all children elements of a parent. It might be useful to target all paragraphs in a particular section.

    23. How do you select all links within a div with the class of sidebar?
    .sidebar div{}

    24. What is a pseudo selector?
    A pseudo-class is a selector that specifies a special state of an element (hover, visited)

    25. What do we use the change the spacing between lines?

    26. What do we use to change the spacing between letters?
    letter-spacing

    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
